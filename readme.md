# 将微信读书划线/想法等信息同步到 Notion

## 项目说明

- 支持将微信读书划线/想法同步到 Notion
- 通过 github actions 每天定时同步
- 同步微信读书部分元数据信息

## 致谢
我是基于下面这个库，因为notion更新了api规则，所以我做了更新，为了自用

## 2026 版本升级说明 (Notion API 2025-09-03 适配)

本项目已针对 Notion API `2025-09-03` 版本进行了全面升级，并优化了同步逻辑：

1.  **Notion API 适配**：
    *   全面支持 Notion 新版 API (Version 2025-09-03)。
    *   自动处理新的 `data_source_id` 机制，确保数据库操作兼容性。

2.  **同步逻辑优化**：
    *   **仅同步有划线的书籍**：自动过滤掉没有划线（笔记）的书籍，保持 Notion 数据库整洁。
    *   **智能同步与清理**：
        *   **自动清理**：如果书籍在微信读书中已删除或不再有划线，程序会自动归档（删除）Notion 中对应的页面。
        *   **增量更新**：对于已存在的书籍，仅更新阅读进度、状态等属性，避免重复创建。
    *   **性能提升**：引入本地缓存机制，大幅减少 API 调用次数，加快同步速度。
